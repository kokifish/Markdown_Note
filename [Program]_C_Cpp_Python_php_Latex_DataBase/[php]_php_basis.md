# php

- PHP: Hypertext Preprocessor(超级文本预处理语言)
- 通用开源脚本语言
- PHP 脚本在服务器上执行
- 最流行的网站开发语言

PHP执行过程（Apache）

浏览器找到服务器Apache，Apache访问a.php，把脚本程序发送给PHP处理程序，PHP处理程序查看该脚本是否有数据库交互，有则与后端数据库交互，然后返回的是HTML语言，给浏览器呈现出来。
